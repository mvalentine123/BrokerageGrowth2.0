## Video Optimization Tips

1. **Compress videos** for web delivery using tools like HandBrake or FFmpeg
2. **Use appropriate bitrate**: 2-5 Mbps for 1080p, 1-3 Mbps for 720p
3. **Create poster images**: Export first frame or create custom thumbnail (1920x1080 JPG)
4. **Test autoplay**: Ensure videos work on Safari, Chrome, and Firefox
5. **Consider WebM format**: Create WebM versions as fallback for better browser compatibility
